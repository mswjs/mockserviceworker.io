**Request namespace** is an object that simplifies declaration of [request handlers](/docs/concepts/request-handler).

MSW comes with the following standard request namespaces:

| Namespace                      | Description                                                              |
| ------------------------------ | ------------------------------------------------------------------------ |
| [`http`](/docs/api/http)       | Namespace for [describing RESTful APIs](/docs/network-behavior/rest).    |
| [`graphql`](/docs/api/graphql) | Namespace for [describing GraphQL APIs](/docs/network-behavior/graphql). |

All namespaces can be imported from the `msw` package and used within the same network description.

```js
// handlers.js
import { http, graphql } from 'msw'

export const handlers = [
  http.get('/user/:id', resolver),
  graphql.mutation('SignIn', resolver),
]
```
